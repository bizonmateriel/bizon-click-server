MÉMO PROJET — CRM MAILING BIZON MATÉRIEL / GOOGLE SHEETS + SERVEUR RENDER

CONTEXTE GÉNÉRAL
- Projet CRM/mailing sur Google Sheets + Apps Script.
- Objectif : envoyer des mails prospects depuis Google Sheets, avec 2 boutons dans le mail :
  1) "Occasions disponibles"
  2) "Être recontacté"
- Problème initial : la Web App Google Apps Script utilisée pour gérer les clics ouvrait souvent une page "impossible d’ouvrir le fichier / la page pour le moment".
- Conclusion prise : on garde Google Sheets pour le CRM et l’envoi des mails, mais on déporte la gestion des clics vers un serveur Node.js hébergé sur Render.
- Solution actuelle fonctionnelle : boutons du mail branchés sur le serveur Render, qui envoie les mails d’alerte via Brevo.
